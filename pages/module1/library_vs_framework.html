Library vs Framework

The key difference between a library and a framework lies in the control and structure they provide for your code:

Library

  • Usage: A library is a collection of pre-written code that you can call upon to solve specific problems or perform specific tasks.
  • Control: You are in control of the flow of the application. You call the library's functions when needed.
  • Example
    • jQuery (manipulate the DOM)
    • Lodash (utility functions)
    • Axios (HTTP requests)
